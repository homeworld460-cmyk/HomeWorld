# HomeWorld – Week 1 Tasks (Act 1: Lone Wanderer)

When you open the Editor, use this list. Order: PCG + GAS first, then mission flow.

## Tech

- **PCG:** One forest biome (proc-gen), load on-demand. To create it programmatically: run `Content/Python/create_pcg_forest.py` from the Editor (Tools → Execute Python Script); see [PCG_FOREST_SETUP.md](PCG_FOREST_SETUP.md). Add Quixel tree/rock paths to `Content/Python/pcg_forest_config.json` and re-run if desired.
- **GAS:** Implement 3 skills for the survivor.
- **Building:** Basic snap/placement (or placeholder).

## Content

- First playable map: crash landing → scout → boss → claim home.
- Use FAB char + Quixel where applicable.

## Missions

- **Mission 1:** Crash + orient.
- **Mission 2:** Scout biomes / find resources.
- **Mission 3:** Boss + claim home.

## Art/Story

- Whimsical concept tone; emotional beat: isolation → determination (reference only; no asset creation required in this task list).

## Playtest

- **Goal:** "Survive 3 missions" end-to-end.
